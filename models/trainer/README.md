# Trainer plane — build files

3D-printable parts for the **Phase-1 foamboard trainer**. The trainer was built by following a
YouTube tutorial — these `.obj` files are the printed components used in that build (landing gear,
motor mount, spar connectors).

> **One deviation from the tutorial:** the tutorial uses a plain receiver, but this build uses the
> project's **CoreWing F405 WING V2 flight controller** instead (same FC family as the F-35B, so the
> trainer doubles as FC/RX setup validation). Everything else follows the tutorial.

## Tutorial
- **Video:** https://www.youtube.com/watch?v=fMezISrCoNE
- **Backup (in case the video disappears):** https://drive.google.com/file/d/1ASnp0k2xXMjiM-009wzewt9H2370stzq/view?usp=sharing
